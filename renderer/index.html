How to Create Your Own Excel File

If you want to manually prepare an Excel file instead of using the app to add products, create a data/products.xlsx file with the following column headers:

ColumnTypeDescription
idtextUnique product identifier
barcodetextProduct barcode (UPC/EAN)
nametextProduct name
categorytextCategory
purchase_pricenumberUnit purchase cost
sale_pricenumberUnit sale price
stocknumberUnits in inventory
sold_by_boxtrue/falseWhether also sold in boxes
box_unitsnumberUnits per box
box_purchase_pricenumberBox purchase cost
box_sale_pricenumberBox sale price

The app auto-creates this file on first launch — you usually don't need to create one manually. Use Add Product in the Inventory tab instead.
